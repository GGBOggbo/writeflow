把公众号内容做成作品，而不是排成一篇稿

主题负责气质，模块负责说服力。一套为公众号内容而生的表达系统，让教程、方案、发布稿和服务页都更有结构、更有质感，也更容易被读完、被记住、被转化。

这不是把 Markdown 转成微信格式这么简单，而是把标题、正文、模块、主题和行动放进同一套标准里，让一篇内容既有气质，也有说服力。

✨ 主要功能

  • • 实时预览：左侧编辑，右侧即时查看排版结果。
  • • 一键复制：轻松复制富文本格式，直接粘贴到公众号后台。
  • • 主题系统：不是换配色，而是切换内容气质和阅读感受。
  • • 高级模块：让封面、数据、对比、步骤、案例、FAQ 和收尾形成统一结构。
  • • 代码与正文共存：技术内容、参数说明、表格和排版模块能保持在同一套秩序里。
  • • API 转换：适合接入自动化工作流，把内容生成和排版串起来。
  • • 图片与分享：支持图片上传、图片模式和长图分发。

🚀 如何使用

🖊️ 基础编辑

  1. 1. 在左侧的编辑区输入或粘贴您的 Markdown 内容。
  2. 2. 在右侧的预览区查看实时渲染的效果。
  3. 3. 点击右上角的"复制"按钮，然后到微信公众号后台粘贴即可。

🔗 API 调用

通过 RESTful API 实现自动化转换和批量处理：

# 转换 Markdown 为微信格式
curl -X POST /api/convert \
  -H "Content-Type: application/json" \
  -d '{"markdown": "# 标题", "theme": "default"}'

🖼️ 图片与分享

编辑器同时支持内容分发相关能力：

  1. 1. 图片上传：粘贴或上传图片后自动插入 Markdown。
  2. 2. 图片模式：一键切换图片背景与展示方式。
  3. 3. 长图分享：将排版结果导出为适合传播的长图。
